# Groove-PHP

1. This is a ported Version of groove-dl to PHP by Check.
   * groove-dl : http://github.com/jacktheripper51/groove-dl
2. Requirement for pc:
   * php 
   * php5-curl
   * Webserver
   * php5-cli //Important for background download
3. Requirement for iOS Devices:
   * Openssh
   * curl
   * lighttpd
   * php
